# Portfolio Website — goranljiljak.com

## Stack
- Astro (static site generator)
- Tailwind CSS for styling
- Markdown for blog posts and project pages

## Site structure
- Home/Hero section
- About + Work Experience (timeline)
- Projects showcase (grid)
- Blog (list + individual post pages)
- Contact section

## Design goals
- Clean, modern, professional
- Fast and mobile-responsive
- Dark or light mode (TBD)

## Deployment
- Netlify, connected to GitHub main branch
- Astro builds automatically on push
